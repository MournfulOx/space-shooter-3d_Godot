# Space Shooter 3D

A fast-paced 3D arcade space shooter built with **Godot Engine 4.6**.  
Dodge incoming meteors, blast through obstacles, and survive as long as you can — your best time is saved automatically.

**[Play on itch.io](https://mournfulox.itch.io/space-shooter-3d)**

---

## Controls

| Action | Keys |
|--------|------|
| Move   | `WASD` / Arrow Keys |
| Shoot  | `Space` |

## Features

- Endless survival gameplay with a live timer
- Procedurally spawned meteors and obstacles
- High score persistence across sessions
- Retro arcade aesthetic with custom fonts and shaders
- Original sound effects and background music

## Built With

- [Godot Engine 4.6](https://godotengine.org/) — Forward Plus renderer
